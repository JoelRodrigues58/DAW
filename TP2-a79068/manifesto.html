Resumo:

Em primeiro lugar, neste trabalho foi desenvolvido um Document Type Defenition com o objetivo de estruturar um ficheiro XML que contém a informação relativa ao presente documento. Este ficheiro XML terá obrigatoriamente de obdecer a todas as regras estabelecidas para poder ser validado.

Seguidamente, sendo que o objetivo era elaborar um ficheiro HTML que representa a página atual, foi necessária a concretização de um ficheiro XSLT que tem como objetivo a transformação do XML em HTML.

Finalmente, de modo a ir de encontro a um outro objetivo paralelo, foi desenvolvido novamente um Document Type Defenition mas agora para estruturar e validar ficheiros da plataforma clav.dglab.gov.pt.
